# python-chatbot

>마르코프 체인을 사용하여 이야기를 할 수록 더 질높은 답장이 나옴

# 개발환경 
1. python 3.5
2. ubuntu 16.04


### 필요한 모듈 
1. bs4
2. requests
3. konlpy
4. json
5. random



### 사용법 

#### 위치 python-chatbot에서 python3 -m http.server --cgi 8080을 치면 당신의 IP주소:8080/cgi-bin/chatbot.py을 통해 접근 가능
